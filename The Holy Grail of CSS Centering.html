O Santo Graal da Centralização com CSS

Chris Coyier publicou recentemente um fluxo de decisão em seu site, CSS Tricks, para demonstrar a quantidade de métodos para centralizar coisas, verticalmente e horizontalmente, usando CSS. Como soluções individuais, elas tem lá seus problemas, contudo, combinadas, são uma ferramenta poderosa para centralizar algo e que funciona entre os variados navegadores, de forma consistente, a partir do IE8.

Vejamos como se faz.

Blocos Iniciais

Se você está criando um site responsivo, provavelmente dimensionará seus elementos em percentuais, então, criemos um elemento .container qualquer:

<div class="container">
  ...
</div>

.container {
  width: 70%;
  height: 70%;
  margin: 40px auto;
  background: red;
}

Para garantir que nossos elementos se estendam apropriadamente, adicionemos o seguinte CSS html, body { width: 100%; height: 100%; }.

Virando a Mesa

Agora, dentro de .container, centralizaremos de forma vertical com o conhecido padrão das tabelas. Isso faz com que elementos block comuns comportem-se como células de tabelas, permitindo centralização vertical:

<div class="container">
  <div class="outer">
    <div class="inner">
      ...
    </div>
  </div>
</div>

.outer {
  display: table;
  width: 100%;
  height: 100%;
}
.inner {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}

Por fim, adicionaremos um elemento centralizado horizontalmente:

<div class="container">
  <div class="outer">
    <div class="inner">
      <div class="centered">
        ...
      </div>
    </div>
  </div>
</div>

.centered {
  position: relative;
  display: inline-block;

  width: 50%;
  padding: 1em;
  background: orange;
  color: white;
}

Dentro deste elemento, você pode colocar qualquer coisa que deseja, incluindo blocos de texto dinâmicos ou elementos com posições absolutas.

Alterações

Para alterar a centralização horizontal de um elemento, basta modificar a propriedade text-align de .inner. Para alterar a centralização vertical, modifique a propriedade vertical-align de .inner.

Concluindo

Esse exemplo parece ter muito código HTML, mas as peças funcionais são apenas três divs: .outer, .inner, e .centered. E a estilização delas é consistente, de forma que você pode usar no código CSS base de seus projetos e nunca se preocupar novamente.

Essa técnica só usa um elemento a mais que a maioria das técnicas de centralização por aí e funciona perfeitamente. Espero que ela se torne a abordagem padrão para centralização de elementos, para que possamos, finalmente, parar de nos preocuparmos em relação a centralização com CSS.
